# Laravel - Base pronto para usar

Despues de clonar:
- Copiar .env.example en .env:
	- `cp .env.example .env`
	- Modificar contenido si es necesario (ej. credenciales de DB)
- Instalar paquetes de composer:
	- `composer install`
- Generar claves de Laravel:
	- `php artisan key:generate`
